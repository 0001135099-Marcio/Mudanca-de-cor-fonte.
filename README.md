Página Interativa: Romeu e Julieta

Um projeto simples em HTML criado para demonstrar a interação entre elementos da página e JavaScript. O projeto apresenta um breve resumo sobre a clássica obra "Romeu e Julieta" de William Shakespeare e permite que o usuário altere o estilo do texto em tempo real clicando em botões.

Funcionalidades
A página contém botões interativos que modificam o estilo dos parágrafos usando JavaScript:

Mudar Tamanho da Fonte: Aumenta a letra do primeiro parágrafo para 35px.

Mudar Cor do Texto: Altera a cor do primeiro parágrafo para vermelho.

Mudar para Negrito: Aplica o estilo negrito (bold) ao segundo parágrafo.

Mudar para Itálico: Aplica o estilo itálico ao segundo parágrafo.

Tecnologias e Conceitos Utilizados
HTML5: Estruturação dos textos, botões e imagem.

JavaScript Inline: Uso do evento onclick diretamente nos botões para disparar ações.

Manipulação do DOM: Uso do método document.getElementById().style para alterar as propriedades CSS (tamanho, cor, peso e estilo da fonte) dinamicamente.

Como Usar
Salve o código em um arquivo com a extensão .html (por exemplo, index.html).

Certifique-se de ter uma imagem chamada Romeu e Julieta.webp salva na mesma pasta que o seu arquivo HTML (ou altere o atributo src da tag <img> para uma imagem que você já tenha).

Dê um duplo clique no arquivo .html para abri-lo em qualquer navegador web (Google Chrome, Firefox, Edge, etc).

Clique nos botões para ver os textos mudando de estilo na hora!
